• Default: Increase the micro version
  • Release Candidate: Add RC before -SNAPSHOT, making it x.x.x-RC-SNAPSHOT
  • Ubuntu like Monthly: yy.MM.dd for branch; yy.MM+1 for new trunk
  • Ubuntu like Monthly with Job Name/ID: Same as Ubuntu like Monthly,
    added trunk job name (before the first underscore "_") to the branch job name,
    e.g. trunk job name is ABC_TRUNK, the newly created branch job will be: ABC_BRANCH_${branch-version}
