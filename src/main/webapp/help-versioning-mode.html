• Default: Increase the micro version
  • -RC-SNAPSHOT: Add RC before -SNAPSHOT, making it x.x.x-RC-SNAPSHOT
  • trunk: yy.MM-SNAPSHOT, branch: yy.MM.dd-SNAPSHOT: yy.MM.dd for branch; yy.MM+1 for new trunk
  • trunk: yy.MM-SNAPSHOT, branch: yy.MM.dd-${first world in Job name}-SNAPSHOT:
    added trunk job name (before the first underscore "_") to the branch job name,
    e.g. trunk job name is ABC_TRUNK, the newly created branch job will be: ABC_BRANCH_${branch-version}
